## Research

**Automated LLM-assisted Invariant Generation and Verification for Smart Contracts** *(MSc Thesis, IST Lisboa)*

Smart contracts are immutable once deployed, vulnerabilities can't be patched and
exploits are permanently recorded on-chain. This thesis explores using LLMs to
automatically reason about smart contract code, generate invariants, and verify them
with invariant-checking tools. The goal: reduce the manual burden on developers and
improve the thoroughness of pre-deployment security analysis.

*Areas touched: formal verification · LLM reasoning · smart contract security · DeFi · DAOs*
